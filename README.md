# 🛡️ Bóveda Digital - Gestor de Contactos

**Fecha de entrega:** 16 de abril de 2026

## 📝 Descripción
Este programa es una agenda de contactos robusta desarrollada en **C++** utilizando Programación Orientada a Objetos (POO). Permite al usuario agregar, buscar y visualizar contactos, garantizando que la información se guarde de forma permanente en un archivo de texto (`datos_agenda.txt`).

## 🛠️ Instrucciones de Compilación y Ejecución
Para compilar el proyecto en cualquier terminal con GCC, utiliza el siguiente comando:

```bash
g++ main.cpp Agenda.cpp Contacto.cpp -o BovedaDigital
